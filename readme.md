# windowAIChat

一个专注于 AI 对话的桌面应用，将智能聊天与文档编辑深度整合。基于 Electron 构建，提供原生桌面体验。

## 特点

- **手动挡 / 自动挡**：灵活的上下文管理方式，可精确控制也可自动化处理
- **对话分支**：探索不同思路，随时切换对话方向
- **本地优先**：所有数据存储在本地，隐私安全
- **多模型**：支持 OpenAI、Gemini 等多种 AI 模型
- **文档集成**：边聊边写，AI 辅助文档创作

## 核心功能

### AI 聊天模块

这是应用的核心，提供强大而灵活的对话能力：

#### 上下文管理：手动挡 || 自动挡

- **手动挡模式**
  - 自己选择要发送的历史消息
  - 精确控制每次请求的上下文内容
  - 适合需要精细调整对话上下文的场景
  
- **自动挡模式**
  - 智能压缩历史对话，保持关键信息
  - 省心省力，让 AI 决定上下文

#### 对话分支管理

- 多分支对话树结构
- 支持在任意节点对消息进行修改，删除，或重新生成
- 随时切换分支，对比不同的回答

#### 临时上下文

- 将文件内容或者提示词作为临时上下文发送给 AI
- 不占用历史对话记录

#### 多模型支持

- **OpenAI 兼容接口**：支持 OpenAI API 及其兼容服务
- **Google Gemini**：直接接入 Gemini 模型
- **易扩展**：基于适配器架构，方便接入新模型

#### 对话持久化

- 所有对话自动保存到本地数据库
- 按时间线浏览历史会话
- 快速搜索和定位历史对话

### 提示词模板系统

- **模板管理**：创建、编辑、删除提示词卡片
- **快速应用**：一键将模板应用到当前对话
- **独立窗口**：支持独立窗口管理模板，多窗口协同工作
- **预设场景**：内置常用写作场景模板

### 文档编辑

- **Markdown 支持**：GFM 语法、数学公式（KaTeX）、代码高亮
- **DOCX 支持**：导入导出 Word 文档，保持基本格式
- **实时预览**：编辑与预览同步

### 工作区管理

- 本地文件树浏览
- 文件/文件夹的创建、重命名、删除
- 工作区快速切换
- 右键菜单：在资源管理器中打开、复制路径等
